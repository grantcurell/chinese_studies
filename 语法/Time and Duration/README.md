If you have a location it is interchangable with a time word (yesterday, today, etc):
- 主语 + 时间名词
- 时间名词 + 主语
  
Are both correct

### Time Words
时间名词 = when
时长 = duration

### Corrected sentences

### Difference between 等待 and 等
等待 （东西 - 不可以是人）
等 （东西 或 人）
